Module Name: Purchase Order
Description:
Track purchase orders made for the customer. This module works with
the Customer module.

Requirements:
  * Customer Module
  * PO Database Tables

Module Saved Data:
  * Customer name
  * Shipping to name & address (if different from customer's info)
  * Purchase date
  * Item cost
  * Purchase price
  * Discounted price / percent
  * Notes (details)
